Especificacion archivos:

MAKEFILE: Compila y ejecuta codigo, escribiendo make en la consola.
Lab2.c : Codigo escrito en c, el cual al compilarlo crea un archivo a ejecutar.
lab2 : Ejecutable compilado el cual inicia el programa.


Instrucciones de compilacion: 
En consola ubuntu o terminal wsl, dirigirse a la direccion donde se encuentra el archivo Lab2.c, una vez en la ubicacion, en la consola escribir make, de esta forma el
makefile ejecutara el codigo para compilar el archivo y lo ejecutara. En la consola se indicaran las filas generadas, despues de 3 seg aprox, se le preguntara por la
carta a jugar. Una vez ingresado el input, se printearan todos las cartas jugadas ese turno y se colocaran en una fila con un numero cercano automaticamente, partiendo
por la carta jugada de menor valor. Finalmente pasado 10 turnos se printeara por consola los puntajes negativos finales y quien fue el ganador.
