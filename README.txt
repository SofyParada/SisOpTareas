Especificacion archivos:

MAKEFILE: Compila y ejecuta codigo, escribiendo make en la consola.
Lab1.c : Codigo escrito en c, el cual al compilarlo crea un archivo a ejecutar.
prueba : Ejecutable compilado el cual inicia el programa.


Instrucciones de compilacion: 
En consola ubuntu o terminal wsl, dirigirse a la direccion donde se encuentra el archivo Lab1.c, antes de ejecutar make se debe determinar la direccion de la carpeta en donde se encuentra los archivos .txt a ordenar. Una vez indicado la direccion en el codigo, el la consola escribir make, de esta forma el makefile ejecutara el codigo para compilar el archivo y lo ejecutara. En la consola indicara que accion desea tomar, 1 para ordenar segun jugadores actuales y 2 para jugadores totales. Una vez ingresado el input, se printearan todos los resultados ordenados por genero y al final los totales. Si se quiere ejecutar denuevo se deben agregar nuevamente los archivos a la carpeta y borrar carpetas creadas en la ejecucion del codigo.
