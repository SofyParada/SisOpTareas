Especificacion archivos:

MAKEFILE: Compila y ejecuta codigo, escribiendo make en la consola.
src : Carpeta que contieen archivos clase para ejecucion.


Instrucciones de compilacion: 
En consola ubuntu o terminal wsl, dirigirse a la direccion donde se encuentra el archivo MAKEFILE, 
una vez en la ubicacion, en la consola escribir make, de esta forma el makefile 
compilara los archivos creando una carpeta build y un ejecutable .jar, el cual es iniciado al mismo tiempo.
 
En la consola se indicaran los valores leidos, luego secuencialmente se ejecutaran los programas para 
buscar el tesoro en orden thread, fork y normal, mostrando el tiempo de ejecucion de cada uno en nanosegundos.
